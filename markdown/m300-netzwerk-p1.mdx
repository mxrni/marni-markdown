# M300 Netzwerk ~Plattformübergreifende Dienste in ein Netzwerk integrieren~

## Hardware

- **Router**: Ein Gerät, das den Datenverkehr in einem Netzwerk weiterleitet.
- **Modem**: Ein Gerät, das digitale Signale in analoge umwandelt und umgekehrt, um eine Internetverbindung herzustellen.
- **Modulator**: Ein Gerät, das ein analoges Signal aus einem digitalen erzeugt.
- **Demodulator**: Ein Gerät, das ein digitales Signal aus einem analogen erzeugt.
- **Firewall**: Ein Sicherheitsgerät, das den Datenverkehr zwischen einem internen Netzwerk und dem Internet überwacht und filtert.
- **Switches**: Ein Gerät, das den Datenverkehr in einem lokalen Netzwerk (LAN) steuert.
- **Hub**: Ein Gerät, das den Datenverkehr in einem Netzwerk weiterleitet.

## Kabel, Endgeräte, PCs, Server etc.

- **Kabel**: Ein Medium zur Übertragung von Daten zwischen Geräten in einem Netzwerk.
- **Endgeräte**: Geräte, die am Netzwerk angeschlossen sind und Daten senden und empfangen können.
- **PCs**: Personal Computer, die als Endgeräte oder Server in einem Netzwerk fungieren können.
- **Server**: Ein Gerät, das Dienste und Ressourcen für andere Geräte in einem Netzwerk bereitstellt.
- **Access Point**: Ein Gerät, das drahtlose Geräte mit einem kabelgebundenen Netzwerk verbindet.
- **Repeater**: Ein Gerät, das das Funksignal in einem drahtlosen Netzwerk verstärkt.
- **Gateway**: Ein Gerät, das den Datenverkehr zwischen verschiedenen Netzwerken (z.B. LAN und Internet) steuert.

## ISP – Internet Service Provider

- **Internetdienstanbieter**: Ein Unternehmen, das Internetdienste für Endbenutzer bereitstellt.
- **Abdeckung Standort**: Die Verfügbarkeit von Internetdiensten an einem bestimmten Standort.
- **Kosten**: Die Gebühren, die ein Benutzer für den Zugang zu Internetdiensten zahlen muss.
- **Geschwindigkeit**: Die Datenübertragungsrate eines Internetanschlusses.
- **Bandbreite**: Die maximale Datenmenge, die über einen Internetanschluss übertragen werden kann.
- **Up/Down**: Die Upload- und Download-Geschwindigkeiten eines Internetanschlusses.
- **SLA**: Service Level Agreement, ein Vertrag zwischen einem ISP und einem Benutzer, der die Qualität und Verfügbarkeit von Internetdiensten festlegt.

## Situationsanalyse

- **Wie viele User**: Die Anzahl der Benutzer, die das Netzwerk verwenden.
- **Wie viele Geräte**: Die Anzahl der Geräte, die mit dem Netzwerk verbunden sind.
- **Anforderungen (Was machen User/Geräte)**: Die spezifischen Anforderungen, die Benutzer oder Geräte an das Netzwerk stellen.
- **Wie kommen die Geräte ins Netzwerk**: Die Methoden, mit denen Geräte mit dem Netzwerk verbunden werden.
- **Anzahl Netzte**: Die Anzahl der Netzwerke, die miteinander verbunden sind.

## Backup / Fallback / Zweite Leitung

- **Backup**: Eine alternative Verbindung oder Dienstleistung, die im Falle eines Ausfalls des Hauptsystems verwendet werden kann.
- **Fallback**: Ein Mechanismus, der automatisch auf eine alternative Verbindung oder Dienstleistung umschaltet, wenn das Hauptsystem ausfällt.
- **Zweite Leitung**: Eine zusätzliche Verbindung oder Dienstleistung, die als Backup für das Hauptsystem verwendet wird.

## Technologie

- **GSM**: Global System for Mobile Communications, ein Mobilfunkstandard.
- **Funk**: Ein drahtloses Kommunikationsmedium, das elektromagnetische Wellen verwendet.
- **5G**: Der fünfte Mobilfunkstandard, der höhere Datenübertragungsraten und geringere Latenzzeiten bietet.
- **4G**: Der vierte Mobilfunkstandard, der eine schnellere Datenübertragung und eine bessere Netzabdeckung bietet.
- **LTE**: Long Term Evolution, ein Standard für drahtlose Breitbandkommunikation.
- **3G**: Der dritte Mobilfunkstandard, der höhere Datenübertragungsraten als 2G bietet.
- **H+**: Eine verbesserte Version von 3G, die noch höhere Datenübertragungsraten bietet.
- **2G**: Der zweite Mobilfunkstandard, der hauptsächlich für Sprachkommunikation verwendet wird.
- **EDGE**: Enhanced Data rates for GSM Evolution, ein Standard für die Übertragung von Daten über GSM-Netze.
- **WLAN**: Wireless Local Area Network, ein drahtloses Netzwerk, das auf IEEE 802.11-Standards basiert.
- **MIMO**: Multiple Input Multiple Output, eine Technologie, die die Datenübertragungsrate in drahtlosen Netzwerken erhöht.
- **Multi MIMO**: Eine erweiterte Version von MIMO, die mehrere Antennen in drahtlosen Geräten verwendet.
  ![mimo image](3lehrjahr/mimo.png)
- **Höhere Übertragungsraten**: Eine höhere Datenübertragungsrate in drahtlosen Netzwerken.
- **Grössere Empfangsleistung**: Eine bessere Empfangsqualität in drahtlosen Netzwerken.
- **Bessere Verbindungqualität**: Eine höhere Zuverlässigkeit und Stabilität der Verbindung in drahtlosen Netzwerken.
- **Kanäle**: Die Frequenzbänder, die in drahtlosen Netzwerken verwendet werden.
- **Frequenzen**: Die Frequenzbänder, die in drahtlosen Netzwerken verwendet werden.
- **2,4 GHz, 5 GHz**: Die Frequenzbänder, die in drahtlosen Netzwerken verwendet werden.
- **Einfluss**: Die Auswirkungen von Faktoren wie Interferenzen und Reichweite auf die Leistung eines drahtlosen Netzwerks.
- **Bandbreit**: Die maximale Datenmenge, die über ein Netzwerk übertragen werden kann.
- **Interferenzen**: Die Störungen, die die Leistung eines drahtlosen Netzwerks beeinträchtigen können.
- **Reichweite**: Die maximale Entfernung, über die ein drahtloses Netzwerk funktionieren kann.
- **Beeinflussung durch Wände, Stahlträger, Glasscheiben,...**: Die Auswirkungen von physischen Hindernissen auf die Leistung eines drahtlosen Netzwerks.

## Kabel

- **Kupfer**: Ein Material, das häufig für die Herstellung von Kabeln verwendet wird.
- **RJ45**: Ein Standardstecker für Ethernet-Kabel.
- **EthernetPatchkabel**: Ein Kabel, das zur Verbindung von Geräten in einem Ethernet-Netzwerk verwendet wird.
- **Netwerkkabel**: Ein Kabel, das zur Verbindung von Geräten in einem Netzwerk verwendet wird.
- **Twisted-Pair**: Ein Typ von Kabel, das aus zwei verdrillten Kupferdrähten besteht.
- **Abschirmungen**: Schutzvorrichtungen, die Kabel vor elektromagnetischen Störungen schützen.
  ![shielded image](3lehrjahr/shielded.jpg)
- **DSL**: Digital Subscriber Line, eine Technologie für die Übertragung von Daten über Kupferkabel.
- **ADSL**: Asymmetric Digital Subscriber Line, eine Variante von DSL, die unterschiedliche Upload- und Download-Geschwindigkeiten bietet.
- **VDSL**: Very High Bitrate Digital Subscriber Line, eine Variante von DSL, die höhere Datenübertragungsraten bietet.
- **ISDN**: Integrated Services Digital Network, eine digitale Telefonnetztechnologie.
- **Glas**: Ein Material, das für die Herstellung von Glasfasern verwendet wird.
- **LWL**: Lichtwellenleiter, ein Kabel, das für die Übertragung von Lichtsignalen verwendet wird.
- **FTTx**: Fibre to the..., eine Technologie, die Glasfaserkabel bis zum Benutzeranschluss verwendet.

## Signal

- **Duplex**: Die Fähigkeit, gleichzeitig Daten in beide Richtungen zu übertragen.
- **Full-Duplex**: Die Fähigkeit, Daten in beide Richtungen gleichzeitig zu übertragen.
- **Halb-Duplex**: Die Fähigkeit, Daten in beide Richtungen nacheinander zu übertragen.
  ![duplex image](3lehrjahr/duplex.png)
- **Modulieren**: Die Umwandlung von digitalen Daten in ein analoges Signal.
- **Demodulieren**: Die Umwandlung von analogen Signalen in digitale Daten.
- **UDP**: User Datagram Protocol, ein Protokoll für die Übertragung von Daten über ein Netzwerk.
- **Senden ohne Überprüfung**: Die Übertragung von Daten ohne Bestätigung ihrer Ankunft.
- **Schneller**: Die Übertragung von Daten mit höherer Geschwindigkeit.
- **TCP**: Transmission Control Protocol, ein Protokoll für die Übertragung von Daten über ein Netzwerk.
- **Senden mit Überprüfung**: Die Übertragung von Daten mit Bestätigung ihrer Ankunft.
- **Bestätigte Ankunft**: Die Bestätigung, dass Daten sicher und vollständig übertragen wurden.
- **CSMA-CA**: Collision Avoidance, ein Mechanismus zur Vermeidung von Datenkollisionen in einem Netzwerk.
- **CSMA-CD**: Collision Detection, ein Mechanismus zur Erkennung von Datenkollisionen in einem Netzwerk.

## Netzwerke / Zonen

- **LAN**: Local Area Network, ein lokales Netzwerk, das Geräte in einem begrenzten Bereich verbindet.
- **VLANS**: Virtual Local Area Networks, logische Unterteilungen eines physischen LANs.
- **Guest (WLAN)**: Ein drahtloses Netzwerk, das für Gäste oder Besucher zur Verfügung steht.
- **DMZ**: Demilitarized Zone, ein Netzwerksegment, das außerhalb des internen Netzwerks liegt und für öffentliche Dienste verwendet wird.
- **Von aussen erreichbar**: Die Fähigkeit, auf ein Netzwerk von außen zuzugreifen.
- **Webserver**: Ein Server, der Webseiten und andere Webinhalte bereitstellt.
- **WAN**: Wide Area Network, ein Netzwerk, das Geräte in verschiedenen geografischen Standorten verbindet.
- **Restliche Netze**: Andere Netzwerke, die nicht in die oben genannten Kategorien fallen.

## Netzwerkplan / - Topologie

- **Dokumentiertiert, Protokolliert, Abgesegnet, Abgelegt (Verträge)**: Die Dokumentation und Protokollierung von Netzwerkplänen und -topologien, um sicherzustellen, dass sie den Anforderungen und Vereinbarungen entsprechen.
- **Proxy-Server**: Ein Server, der als Vermittler zwischen einem internen Netzwerk und dem Internet fungiert und den Datenverkehr filtert und überwacht.
